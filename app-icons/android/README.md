# Android App Icons — 乾隆大藏经

Generated by `tool/generate_app_icons.py`, following the
[Google Play icon design specifications](https://developer.android.com/distribute/google-play/resources/icon-design-specifications)
and the adaptive-icon guidelines:

- **Play Store listing icon**: `play-store/ic_launcher_512.png` — 512×512,
  32-bit PNG, full square with no baked rounding or shadow (Play applies the
  20% corner radius and drop shadow dynamically), < 1 MB.
- **Adaptive icon layers** (API 26+): 108dp foreground/background per
  density; all key content inside the 66dp safe-zone circle.
- **Monochrome layer** (API 33+ themed icons): single white-alpha 藏 glyph.
- **Legacy launcher icons** (API < 26): 48dp `ic_launcher.png`
  (rounded-square shape baked in) and `ic_launcher_round.png` (circular).

## Feature graphic

`play-store/feature_graphic_1024x500.{png,jpg}` — Play Store listing feature
graphic, per the
[feature graphic spec](https://support.google.com/googleplay/android-developer/answer/9866151):
1024×500 px, 24-bit, no alpha. The 藏 ensō emblem sits left-of-center with
the title block on the right, leaving the dead-center gutter clear so an
optional promo-video play-button overlay covers no important content; all
text stays well inside the edges. Both PNG (533 KB) and JPEG (q92, 95 KB)
are provided — upload either.

## Install

Copy `res/` over `android/app/src/main/res/` in the Flutter runner. The
manifest already references `@mipmap/ic_launcher`; add
`android:roundIcon="@mipmap/ic_launcher_round"` to `<application>` if not
present.
