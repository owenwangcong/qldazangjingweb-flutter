# iOS App Icons — 乾隆大藏经

Generated by `tool/generate_app_icons.py`, following the
[Apple HIG App Icons](https://developer.apple.com/design/human-interface-guidelines/app-icons) spec:

- Full-bleed square PNGs, **no transparency**, **no pre-rounded corners**
  (the system applies the superellipse mask).
- Key elements (glyph + seal) kept inside the central ~80% so nothing is
  clipped by the corner mask.
- `Icon-App-1024x1024@1x.png` is the App Store marketing icon (opaque RGB).

## Contents

`AppIcon.appiconset/` — complete Xcode asset-catalog icon set with
`Contents.json` (iPhone 20/29/40/60pt @2x/@3x, iPad 20/29/40/76pt @1x/@2x,
83.5pt @2x, App Store 1024pt).

## Install

Replace the existing set in the Flutter runner:

```
ios/Runner/Assets.xcassets/AppIcon.appiconset/  ←  app-icons/ios/AppIcon.appiconset/
```
